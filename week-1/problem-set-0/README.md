## Key Learnings

- `input()` returns string, need explicit conversion to int/float
- `.replace()` for character substitution
- `.strip("$")` cleaner than `.replace("$", "")` for known prefix/suffix
- `f"{value:.2f}"` for 2 decimal places formatting
